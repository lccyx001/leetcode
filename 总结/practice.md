# 如何修炼

## 菜鸟->新手

- 对于简单的模拟方法，在5-10min内写出代码
- 5-10min内写出暴力算法
- 针对题目的不同情况，在脑海中有对应的思路，例如N=2会怎样，N=3会怎样
  
### 如何训练

简单-中等题目，中等题目，思考15-30分钟后再看题解.看高手的解法，通常代码更加简洁。对于难度较大的题目，建议使用纸和笔。

## 新手->进阶

需要掌握这些算法

- 暴力解法
- DP
- DFS
- WFS
- 狄杰斯特拉算法
- 二叉树
- 组合数学
- 逆元
- 位运算
- 二分法
  
同时，需要更快速的做出题目，5分钟内完成简单题，10分钟做出中等简单题
